# hono-htmx-app

## Use

- [Hono](https://hono.dev/)
- [htmx](https://htmx.org/)
- [Deno](https://deno.com/)
- [hyperscript](https://hyperscript.org/)
- [valibot](https://deno.land/x/valibot@v0.20.0)

## Development

```
# Start App
deno task dev

# Lint
deno task lint

# Format
deno task format

# Test
deno task test
```
